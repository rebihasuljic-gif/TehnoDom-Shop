Aplikacija je razvijena kao moderan web shop sa fokusom na čistu arhitekturu i performanse. 
Kompletan izvorni kod je organizovan prema MVC (Model-View-Controller) obrascu radi lakšeg održavanja i preglednosti.

Korištene Tehnologije:
Backend: Node.js i Express.js

Frontend: HTML5, CSS3 (Moderni UI) i Vanilla JavaScript

Baza podataka: PostgreSQL

Organizacija fajlova:
Controllers/ – Serverska logika i upravljanje rutama.

Models/ – Rad sa podacima i upiti ka bazi.

Views/ – Korisnički interfejs i klijentski skriptovi.

Database/ – SQL skripte za kreiranje i popunjavanje baze.
